# Formulário da Missão Horizonte Vermelho - Marte :ringed_planet:

![Home](Home.PNG "Imagem da página inicial")


## Descrição :page_with_curl:
O projeto em questão é um Front-end de um formulário fictício de uma missão de exploração de Marte. <br> Sendo criado para fins de estudo, este projeto teve como objetivo praticar a utilização de bootstrap e sass, assim como de variados elementos de formulários em HTML. :rocket:

***
